modulo 3


Para el ejercicio, creé una página web utilizando HTML, CSS y JavaScript. Utilicé Bootstrap como mi framework de CSS para agilizar el proceso de diseño y hacer que la página sea responsive.

En la página, implementé una barra de navegación que se desplaza junto con el contenido, con un menú desplegable para dispositivos móviles. Utilicé clases de Bootstrap para centrar el encabezado de la página y para crear una sección principal que ocupara toda la anchura de la ventana del navegador en todas las vistas.

Además, implementé una serie de secciones opcionales utilizando el sistema de rejilla de Bootstrap, y agregué enlaces de "Volver arriba" al final de cada sección para mejorar la experiencia del usuario.


modulo 4


Para este ejercicio primero, abrí el archivo script.js y seguí las instrucciones paso a paso para solucionar los problemas en el código. Utilicé Browser Sync para actualizar automáticamente mi navegador y ver los cambios mientras trabajaba en el código. Una vez que completé todas las correcciones y el código funcionaba correctamente, pude ver los saludos impresos en la consola del navegador.
